【宝塔面板使用心得】L00 安装服务器操作系统

2022-01-03

本文共1.5k字，大约需要阅读5分钟。

本系列心得序号以L开头是Linux面板，以W开头是Windows面板。

与桌面操作系统不同，服务器操作系统一般是安装在大型计算机上，用于提供例如Web、应用、数据库等特定服务的操作系统。

宝塔面板可以安装在Linux和Windows系统的服务器上。Linux版支持CentOS 7.1+ (Ubuntu16.04+.、Debian9.0+)，Windows版仅支持Windows Server 2008/2012/2016/2019 64位。

本系列心得基于在VMware Workstation 16.x虚拟机运行的Ubuntu20.04，4核，内存4G，硬盘80G上讲解，并安装Linux 7.8.0版，面板部分内容由于机型与版本原因会有所出入，请读者自行甄别。

安装VMware

打开VMware官网，根据你自己系统的类型，点击页面最下方的立即下载。

然后点开安装程序，开始安装。

同意许可协议。

此处建议勾选增强型键盘驱动程序。

单击“下一步”即可。

单击“安装”，并等待安装进度条。

你可以看到安装已经完毕，可以点击许可证输入密钥或者点击完成后稍后输入密钥，这里点击许可证直接输入密钥。

VMware Workstation 作为虚拟机软件中十分经典而且流行的一款，十分建议各位购买正版软件，支持正版。但是，由于学生的经济实力有限或者你只想试用这款软件，往往你不会去购买正版软件（这还真是可惜），而是使用搜索引擎得到VMware Workstation的破解密钥。

笔者并不会在这里列出那些密钥。

输入密钥并点击“输入”。

单击“完成”，结束安装。

Oops!它提示要重启了！笔者建议各位重启自己的电脑，因为VMware Workstation需要一些系统服务支持才能够正常运行。请确认别的工作文件已经保存，然后点击“是”，笔者在这里等待大家几分钟。

安装Linux

欢迎回到本系列心得！虚拟机准备好了，现在我们来安装操作系统吧！

Linux的发行版本诸多，这里以算法竞赛、电子数据取证竞赛中都经常用到的一个版本Ubuntu来安装。Ubuntu基于Debian，对各类硬件的支持比较完善，而且界面友好、上手容易，最大的优点是Ubuntu的所有发行版本都免费提供。

下载Ubuntu镜像

打开Ubuntu官网，你会发现页面上方列出了许多Ubuntu的发行版本，这里以Ubuntu 20.04.3 LTS (Focal Fossa)为例，给大家讲解下安装操作系统的方法。

点开对应版本的页面，你会发现有两个版本的下载页面。第一个版本是Desktop image，这是桌面版本，经常用于个人电脑；第二个版本是Server install image，即服务器版本。这里，选择服务器版本下载，点击右边的超链接即可下载。

如果你有想尝试使用Ubuntu系统的想法，可以在看完本系列心得后自行下载桌面版本安装使用。

创建虚拟机

现在，我们点开VMware Workstation，你看到了主页选项卡，现在点击“创建新的虚拟机”。

选择“自定义(高级)(C)”，并点击“下一步”。

默认Workstation 16.x即可。

请选择“稍后安装操作系统(S)。”。

请将系统选为Linux—Ubuntu 64 位。

你可以自行选择虚拟机名称和它的位置，这不影响使用。

你可以使用默认的处理器配置，也可以和本心得一样，设为2个处理器每个处理器2核的4核配置。

内存可以是推荐的4GB即4096MB，本心得也用的是4GB。

默认NAT模式即可。

推荐的 LSI Logic 即可。

推荐的 SCSI 即可。

选择“创建新虚拟磁盘”。

推荐20GB，你也可以和本心得一样改成80GB。

这里的名称不影响使用，默认即可。

已经准备好创建虚拟机，但是不要急着点完成，这里点击“自定义硬件”。

选择“新 CD/DVD (SATA)”，选择“使用 ISO 映像文件”，浏览选中本节刚开始下载的 Ubuntu 20.04 的服务器镜像，然后单击“关闭”。

现在可以点击“完成”了。

安装操作系统

现在，你已经看到虚拟机的页面，单击“开启此虚拟机”

当安装程序的光标在“English”上时，敲回车进行选择。

这里它提示我们有更新，直接选择“Continue without updating”，不要更新。

这里按上下键切换选择项，在Layout这里按空格弹出菜单，上下选择“Chinese”，你会发现Variant现在也是“Chinese”，选中“Done”确认即可。

网络连接这里直接确认过去即可。

代理这里直接确认过去跳过即可。

镜像源这里直接默认即可。

磁盘按默认的整个磁盘安装即可。

存储设置这里直接默认即可。这里，它提示是否确认设置，选择“Continue”确认设置。

资料配置这里按你自己的需要填写。

建议安装OpenSSH，按空格选中。

一些服务器环境，建议不要安装，我们将在宝塔面板中对服务器环境进行安装和配置。确认过后我们等待安装即可。

现在，已经安装完了，选择“Reboot Now”重新启动系统。

这个页面直接敲回车即可，VMware已经帮你把镜像自动移除了。

当你看到login或者静止不动，就说明已经加载完毕，可以输入刚刚设置的用户名和密码登录了。

现在，你已经安装完毕了！可以点击VMware Workstation的我已完成安装了。
